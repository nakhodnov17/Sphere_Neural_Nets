# Способы обучения RNN
1. Предсказываем по одному элементу:
```python
o, h = LSTM(x)
loss = mse(o[-1], y[0])
o = o[[-1]]
for i in range(k - 1):
    # teacher forcing: с вероятностью 0.9 передаём - y[1], а с вероятностью 0.1 - o
    o, h = LSTM(0.9 * y[i] + 0.1 * o) 
    loss += mse(o, y[i+1])
loss.backward()
```

2. Предсказываем всё сразу, а loss считаем сдвигая последовательность, но так делать не всегда хорошо, т.к. последовательности могут быть слишком длинными
```python
o = LSTM(X)
loss = mse(o[:-1], x[1:])
```
